# Sistema de Cadastro de Usuários

front-end:
(https://werthercrisley.github.io/Cadastro-De-Usuarios-Front/)

back-end:
(https://github.com/WertherCrisley/Cadastro-De-Usuarios-BackEnd)

Um sistema completo de cadastro de usuários desenvolvido com React + Vite no frontend e Node.js + Prisma no backend, permitindo realizar operações de cadastro, consulta, edição e exclusão de usuários utilizando um banco de dados.

## Tecnologias Utilizadas

### Frontend
- React
- Vite
- React Router DOM
- Axios
- Bootstrap Icons
- CSS

### Backend
- Node.js
- Express
- Prisma ORM

### Banco de Dados
- MongoDb

## Funcionalidades

- Cadastro de usuários
- Listagem de usuários
- Atualização de usuários
- Exclusão de usuários
- Integração entre frontend e backend
- Persistência dos dados em banco de dados
